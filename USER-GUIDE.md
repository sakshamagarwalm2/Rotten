# Rotten - User Guide

## 🚀 Quick Start

### 1. Upload Your Files
Simply click and select your files - they'll upload automatically!

**DOCX File** (Required)
- Click "Upload DOCX" box
- Select your Word document
- Wait for automatic upload

**Background Image** (Optional)
- Click "Background Image" box
- Select PNG, JPG, or JPEG
- See it appear in preview instantly

## 🎨 Customize Your Presentation

### Text Sizes

**Heading Font Size** (16-60pt)
- Use for main titles and section headings
- Default: 32pt
- Recommended: 28-40pt for readability

**Content Font Size** (12-48pt)
- Use for questions and options
- Default: 20pt
- Recommended: 18-24pt for presentations

### Spacing

**Question Gap** (5-50pt)
- Space between questions
- Default: 15pt
- More space = easier to read
- Less space = more content per slide

**Line Spacing** (1.0-3.0x)
- Height between lines
- Default: 1.5x
- 1.0x = Compact
- 2.0x = Spacious

### Colors

**Heading Color**
- Color for main headings
- Default: Dark Gray (#1a1a1a)
- Tip: Use dark colors for contrast

**Question/Option Color**
- Color for question text and options
- Default: Black (#000000)
- Tip: Keep it readable

**Year Tag Color**
- Color for year badges (e.g., "SSC CGL 2023")
- Default: Red (#e02424)
- Tip: Use bright colors to stand out

**Answer Color**
- Color for correct answer highlighting
- Default: Green (#16a34a)
- Tip: Use contrasting color

### Display Options

**Show Answer**
- ☑️ Checked: Shows correct answer on slides
- ☐ Unchecked: Hides answers (for quizzes)

**Show Bullet Points**
- ☑️ Checked: Shows bullets before options
- ☐ Unchecked: Plain text options

**Bullet Style** (when enabled)
- **● Disc**: Filled circle (default)
- **○ Circle**: Hollow circle
- **■ Square**: Filled square
- **1. Numbers**: Numbered list
- **None**: No bullets (plain text)

## 🎯 Best Practices

### For Readability
✅ Use heading size 28-40pt
✅ Use content size 18-24pt
✅ Set line spacing to 1.5x or higher
✅ Use high contrast colors
✅ Add sufficient question gap (15-20pt)

### For Professional Look
✅ Upload high-quality background (1920x1080)
✅ Use consistent color scheme
✅ Enable bullet points for options
✅ Show answers for study materials
✅ Hide answers for quiz presentations

### For Performance
✅ Keep DOCX files under 10 MB
✅ Optimize background images
✅ Close preview when adjusting many settings
✅ Generate only when settings are final

## 💡 Tips & Tricks

### Tip 1: Preview Before Generate
Always toggle preview ON before generating to verify:
- Background image looks good
- Colors are readable
- Layout is correct

### Tip 2: Use Color Contrast
- Dark text on light background
- Light text on dark background
- Test readability in preview

### Tip 3: Adjust for Audience
- **Large audience**: Bigger fonts (32pt+)
- **Small group**: Normal fonts (20-24pt)
- **Printed handouts**: Smaller fonts (16-18pt)

### Tip 4: Bullet Point Selection
- **Disc**: General purpose, professional
- **Numbers**: Sequential steps, rankings
- **Square**: Modern, bold look
- **None**: Clean, minimal design

### Tip 5: Spacing for Readability
- More questions per slide: Lower gap (10pt)
- Fewer questions per slide: Higher gap (20pt+)
- Dense content: Lower line spacing (1.2x)
- Sparse content: Higher line spacing (2.0x)

## ⚠️ Common Issues

### Issue: Upload Not Working
**Solution**: 
- Check file size (DOCX < 20MB, Image < 10MB)
- Ensure correct file format (.docx, .png, .jpg, .jpeg)
- Wait for previous upload to complete

### Issue: Background Not Showing
**Solution**:
- Toggle preview ON to see background
- Check if image uploaded successfully
- Try a different image format
- Ensure image is not corrupted

### Issue: Preview Not Updating
**Solution**:
- Toggle preview OFF and ON again
- Refresh the page
- Re-upload files

### Issue: Generate Button Disabled
**Solution**:
- Upload DOCX file first
- Wait for upload to complete
- Check for error messages
