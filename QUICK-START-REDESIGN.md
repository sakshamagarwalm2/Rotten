## 🎯 Key Features

### **1. Toggle Preview**
- Click the eye icon button (top-right of settings)
- **Show**: Settings panel is compact (400px), preview visible
- **Hide**: Settings panel expands to full width with 3-column grid

### **2. File Upload**
- **Dashed border boxes** for DOCX and background image
- **Hover**: Border turns black, background turns gray
- **Auto-upload**: Files upload automatically when selected
- **Status**: Shows file name and size when uploaded

### **3. Customization Sections**

**Files** (Light gray card):
- DOCX document upload
- Background image upload

**Text Sizes** (White card):
- Heading font size (16-60pt)

**Spacing** (White card):
- Question gap slider (5-50pt)
- Line spacing slider (1.0-3.0x)

**Colors** (White card):
- Heading color picker
- Question/option color picker
- Year tag color picker
- Answer color picker

**Display Options** (White card):
- Show answer checkbox

- **Black button** at the bottom
- **Full width** with download icon

## 🎨 Design Highlights

### **Colors**
- **Primary**: Black (#111111) - All CTAs and headlines
- **Canvas**: White (#ffffff) - Page background
- **Surface**: Light gray (#f5f5f5) - Feature cards
- **Text**: Dark gray (#374151) - Body text
- **Muted**: Medium gray (#6b7280) - Labels

### **Typography**
- **Display**: Inter 600 with negative letter-spacing
- **Body**: Inter 400, 16px
- **Buttons**: Inter 600, 14px

### **Spacing**
- **Sections**: 96px vertical
- **Cards**: 32px padding
- **Gutters**: 24px between items

### **Radius**
- **Buttons**: 8px
- **Cards**: 12px
- **Pills**: Full circle

## 📱 Responsive Design

### **Desktop** (>1024px)
- Settings: 3-column grid (preview hidden)
- Settings: 400px width (preview visible)

### **Tablet** (768-1024px)
- Settings: 2-column grid (preview hidden)
- Settings: Full width (preview visible)

- Settings: 1-column
- Preview: Below settings

## 🎯 Usage Flow

### **Step 1: Upload Files**
1. Click "DOCX Document" box
2. Select your Word file
3. (Optional) Click "Background Image" box
4. Select your image
5. Files upload automatically

### **Step 2: Customize**
1. Adjust text sizes
2. Set spacing preferences
3. Choose colors
4. Configure display options

### **Step 3: Preview** (Optional)
1. Toggle preview on/off
2. See your background image
3. Verify layout

### **Step 4: Generate**
1. Click "Generate PowerPoint"
2. Wait for processing
3. Download automatically

## 🔧 Customization Tips

### **For Professional Look**
- Use black/dark gray for headings
- Use medium gray for body text
- Enable bullet points with disc style
- Show answers for study materials

### **For Quiz Presentations**
- Use bright colors for year tags
- Hide answers
- Use numbered bullet style
- High contrast colors

### **For Training Slides**
- Larger heading size (36-40pt)
- More line spacing (2.0x)
- Show bullet points
- Professional color scheme

## 🐛 Troubleshooting

### **Issue: Preview Not Showing**
**Solution**: Click the eye icon to toggle preview on

### **Issue: Generate Button Disabled**
**Solution**: Upload DOCX file first (auto-uploads when selected)

### **Issue: Background Not Visible**
**Solution**: 
- Make sure preview is toggled on
- Check if image uploaded successfully
- Try a different image format

### **Issue: Colors Not Changing**
**Solution**:
- Click outside color picker to save
- Verify hex code format (#RRGGBB)
- Try using preset colors
